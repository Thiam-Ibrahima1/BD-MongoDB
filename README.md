# API d'Authentification JWT et Gestion de Tâches

Une API RESTful sécurisée construite avec Node.js, Express et JWT pour l'authentification des utilisateurs et la gestion de tâches.

## Fonctionnalités

- **Inscription et Connexion Utilisateur** avec hachage de mot de passe (bcryptjs)
- **Authentification JWT** pour sécuriser les endpoints
- **Gestion CRUD complète des Tâches** (Create, Read, Update, Delete)
- Stockage persistant des données dans des fichiers JSON
- Architecture asynchrone pour de meilleures performances
